The Nike Sales Data Dashboard provides an overview of sales performance using key metrics and interactive charts. It displays Total Revenue (4.34M), Total Profit (3.44M), Profit Margin (79.19%), Total Units Sold (2K), and Total Discount (525.01). •	The Nike Sales Data Dashboard provides a comprehensive overview of sales performance by tracking revenue, profit, units sold, profit margin, and discounts.
